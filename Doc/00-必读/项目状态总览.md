## ?? Project Basic Information

### Project Name
**SceneTodo** (formerly TodoOverlayApp) - A practical todo list management application that can display pending tasks on top of specific applications.

### Core Value
Helps users quickly view and manage related tasks while using specific software, solving the pain points of frequently switching between work and task lists.

### Tech Stack
- **Framework**: .NET 8 + WPF
- **UI Library**: HandyControl
- **Database**: SQLite + Entity Framework Core
- **Icons**: MahApps.Metro.IconPacks

### Project Information
- **Current Version**: v1.5.0
- **Development Stage**: Alpha (Core features under development)
- **Target Platform**: Windows 10/11
- **Repository**: https://github.com/SoloShine/SceneTodo
