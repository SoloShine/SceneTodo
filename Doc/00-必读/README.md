## ?? 文档约定

### 文档状态标识
- ? **已完成** - 功能已实现并测试
- ?? **进行中** - 正在开发或测试
- ?? **部分完成** - 部分实现，有限制
- ? **未实现** - 尚未开发
- ?? **已归档** - 旧版本，仅供参考

### 文档命名规则
- 交接文档: `Session-N到Session-N+1交接文档.md`
- 功能文档: `功能名称.md`
- 测试文档: `测试指南-功能名称.md`
- 归档报告: `Session-N归档完成报告.md`

### 文档更新规则
- 每次会话结束更新交接文档
- 功能实现后更新功能文档
- 测试完成后更新测试文档
- 重大改动更新README和项目状态总览
