## ?? 文档约定

### 文档状态标识
- ? **已完成** - 功能已实现并测试
- ? **进行中** - 正在开发或测试
- ?? **部分完成** - 部分实现，有限制
- ? **未实现** - 尚未开发
- ??? **已归档** - 旧版本，仅供参考

### 文档命名规则
- 交接文档: `交接文档-vX.X.md`
- 功能文档: `功能名称.md`
- 测试文档: `测试指南-功能名称.md`
- 技术文档: `技术主题.md`

### 文档更新规则
- 每次会话结束更新交接文档
- 功能实现后更新功能文档
- 测试完成后更新测试文档
- 重大改动更新README和项目状态总览

## ?? 贡献指南

### 添加新文档
1. 确定文档类型（功能/测试/技术等）
2. 放到对应目录
3. 更新本README的索引
4. 遵循文档命名规则

### 更新现有文档
1. 在文档顶部标注更新日期
2. 保留修改历史
3. 重大修改需要更新README

### 创建会话文档
1. 在`01-会话记录/`下创建新会话目录
2. 包含：交接文档、修改记录、测试指南
3. 更新本README的会话历史
4. 更新项目状态总览
