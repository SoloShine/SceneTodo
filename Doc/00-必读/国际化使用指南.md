# SceneTodo 国际化使用指南

## 概述

SceneTodo 现已支持国际化功能，目前支持中文（简体）和英文两种语言。本文档介绍如何在项目中使用国际化功能。

## 已实现的功能

### 1. 资源文件结构

- `Resources/Strings.resx` - 中文资源文件（默认）
- `Resources/Strings.en.resx` - 英文资源文件

### 2. 语言管理服务

- `Services/LocalizationService.cs` - 核心本地化服务
- 支持动态切换语言
- 支持自动检测系统语言

### 3. 模型类

- `Models/SupportedLanguage.cs` - 支持的语言枚举
- `Models/LanguageSettings.cs` - 语言设置模型
- 已集成到 `Models/AppSettings.cs`

### 4. 语言设置窗口

- `Views/LanguageSettingsWindow.xaml`
- `Views/LanguageSettingsWindow.xaml.cs`

## 在 XAML 中使用国际化

### 方法1: 使用 LocalizationExtension（推荐）

```xaml
<Window xmlns:services="clr-namespace:SceneTodo.Services">
    <Button Content="{services:Localization Common_Save}" />
    <TextBlock Text="{services:Localization Todo_Title}" />
</Window>
```

### 方法2: 使用 Binding

```xaml
<Window xmlns:services="clr-namespace:SceneTodo.Services">
    <TextBlock Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_OK]}" />
</Window>
```

### 方法3: 使用转换器

```xaml
<Window xmlns:services="clr-namespace:SceneTodo.Services">
    <Window.Resources>
        <services:LocalizationConverter x:Key="LocalizationConverter" />
    </Window.Resources>
    
    <Button Content="{Binding Converter={StaticResource LocalizationConverter}, ConverterParameter=Common_Save}" />
</Window>
```

## 在 C# 代码中使用国际化

### 获取本地化字符串

```csharp
using SceneTodo.Services;

// 方法1: 使用索引器
string title = LocalizationService.Instance["Todo_Title"];

// 方法2: 使用 GetString 方法
string message = LocalizationService.Instance.GetString("Message_Success");

// 方法3: 格式化字符串
string formatted = LocalizationService.Instance.GetString("Message_ItemCount", 10);
```

### 切换语言

```csharp
using SceneTodo.Models;
using SceneTodo.Services;

// 切换到英文
LocalizationService.Instance.ChangeLanguage(SupportedLanguage.English);

// 切换到中文
LocalizationService.Instance.ChangeLanguage(SupportedLanguage.ChineseSimplified);

// 自动检测系统语言
LocalizationService.Instance.AutoDetectLanguage();
```

### 监听语言变化

```csharp
LocalizationService.Instance.PropertyChanged += (s, e) =>
{
    if (e.PropertyName == "Item[]")
    {
        // 语言已更改，更新UI
        UpdateUI();
    }
};
```

## 添加新的本地化字符串

### 1. 在 Strings.resx 中添加中文

在 Visual Studio 中：
1. 打开 `Resources/Strings.resx`
2. 添加新的键值对，例如：
   - 名称: `NewFeature_Title`
   - 值: `新功能`

### 2. 在 Strings.en.resx 中添加英文

1. 打开 `Resources/Strings.en.resx`
2. 添加对应的英文翻译：
   - 名称: `NewFeature_Title`
   - 值: `New Feature`

### 3. 在代码中使用

```xaml
<TextBlock Text="{services:Localization NewFeature_Title}" />
```

或者

```csharp
string title = LocalizationService.Instance["NewFeature_Title"];
```

## 资源键命名规范

为了保持一致性，建议遵循以下命名规范：

- **通用操作**: `Common_*` (如 Common_Save, Common_Cancel)
- **窗口标题**: `*Window_Title` (如 MainWindow_Title)
- **待办相关**: `Todo_*` (如 Todo_Title, Todo_Priority)
- **设置相关**: `Settings_*` (如 Settings_Language)
- **消息提示**: `Message_*` (如 Message_Success, Message_Error)
- **标签管理**: `Tags_*` (如 Tags_Management)
- **备份相关**: `Backup_*` (如 Backup_Create)
- **筛选相关**: `Filter_*` (如 Filter_Status)

## 打开语言设置窗口

```csharp
using SceneTodo.Models;
using SceneTodo.Views;

var settings = AppSettings.Load();
var languageWindow = new LanguageSettingsWindow(settings);
if (languageWindow.ShowDialog() == true)
{
    // 用户保存了设置
    MessageBox.Show("语言设置已保存，将在重启后生效");
}
```

## 待迁移的现有文本

项目中存在大量中英文混用的文本，需要逐步迁移到资源文件中。迁移时请注意：

1. 保持功能不变
2. 确保所有语言都有对应翻译
3. 测试切换语言后的界面显示
4. 更新相关文档

## 添加新语言支持

如果将来需要添加新语言（如日文、韩文等）：

1. 在 `Models/SupportedLanguage.cs` 中添加新的枚举值
2. 创建新的资源文件，如 `Resources/Strings.ja.resx`
3. 在 `LocalizationService.cs` 的 `ChangeLanguage` 方法中添加对应的 CultureInfo
4. 更新语言设置窗口以显示新语言

## 注意事项

1. **重启生效**: 语言切换需要重启应用才能完全生效
2. **默认语言**: 默认使用中文（简体）
3. **资源键缺失**: 如果资源键不存在，将显示 `[KeyName]` 格式的占位符
4. **性能**: LocalizationService 是单例模式，不会影响性能
5. **动态绑定**: 使用 Binding 方式可以实现动态语言切换（无需重启）

## 最佳实践

1. **所有用户可见的文本都应该使用本地化**
2. **使用有意义的资源键名称**
3. **保持中英文资源键一致**
4. **为调试信息和日志使用英文**
5. **定期检查遗漏的本地化文本**

## 示例：完整的国际化窗口

```xaml
<hc:Window
    x:Class="SceneTodo.Views.ExampleWindow"
    xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
    xmlns:hc="https://handyorg.github.io/handycontrol"
    xmlns:services="clr-namespace:SceneTodo.Services"
    Title="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Example_Title]}"
    Width="600"
    Height="400">

    <Grid>
        <StackPanel Margin="20">
            <TextBlock Text="{services:Localization Example_Description}" />
            
            <Button 
                Content="{services:Localization Common_Save}"
                Click="SaveButton_Click" />
                
            <Button 
                Content="{services:Localization Common_Cancel}"
                Click="CancelButton_Click" />
        </StackPanel>
    </Grid>
</hc:Window>
```

```csharp
using HandyControl.Controls;
using SceneTodo.Services;
using System.Windows;

namespace SceneTodo.Views
{
    public partial class ExampleWindow : HandyControl.Controls.Window
    {
        public ExampleWindow()
        {
            InitializeComponent();
        }

        private void SaveButton_Click(object sender, RoutedEventArgs e)
        {
            MessageBox.Show(
                LocalizationService.Instance["Message_SaveSuccess"],
                LocalizationService.Instance["Message_Info"],
                MessageBoxButton.OK,
                MessageBoxImage.Information);
        }

        private void CancelButton_Click(object sender, RoutedEventArgs e)
        {
            Close();
        }
    }
}
```
