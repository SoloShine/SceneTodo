# SceneTodo 国际化 - 快速参考

## 🚀 如何使用语言设置

### 用户操作
1. 点击主窗口左下角的 ⚙️ 设置按钮
2. 选择 "语言设置"
3. 在弹出窗口中选择语言（中文/English）
4. 可选：勾选"自动检测系统语言"
5. 点击"保存"
6. **重启应用**使语言生效

## 💻 开发者快速指南

### 在 XAML 中使用国际化

#### 1. 添加命名空间（只需一次）
```xaml
<Window xmlns:services="clr-namespace:SceneTodo.Services">
```

#### 2. 替换硬编码文本
```xaml
<!-- ❌ 错误 -->
<Button Content="保存" />

<!-- ✅ 正确 -->
<Button Content="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_Save]}" />
```

### 在 C# 中使用国际化

#### 1. 添加 using（只需一次）
```csharp
using SceneTodo.Services;
```

#### 2. 获取文本
```csharp
// 简单文本
string text = LocalizationService.Instance["Common_Save"];

// 格式化文本
string msg = LocalizationService.Instance.GetString("Welcome_{0}", userName);
```

#### 3. 在消息框中使用
```csharp
HandyControl.Controls.Growl.Success(
    LocalizationService.Instance["Message_SaveSuccess"]);

HandyControl.Controls.MessageBox.Show(
    LocalizationService.Instance["Message_Info"],
    LocalizationService.Instance["Message_Confirm"]);
```

## 📝 添加新的资源字符串

### 步骤

1. **打开中文资源文件**
   - `Resources/Strings.resx`
   - 添加键值对，例如：
     ```
     名称: Todo_Priority
     值: 优先级
     ```

2. **打开英文资源文件**
   - `Resources/Strings.en.resx`
   - 添加相同键的英文翻译：
     ```
     名称: Todo_Priority
     值: Priority
     ```

3. **在代码中使用**
   ```csharp
   string text = LocalizationService.Instance["Todo_Priority"];
   ```

4. **构建项目**
   - 资源文件会被编译到程序集中

## 🔧 故障排除

### 问题：资源键不存在，显示 [KeyName]
**解决**：
1. 检查 `Strings.resx` 和 `Strings.en.resx` 中是否都有该键
2. 确保键名拼写正确
3. 重新构建项目

### 问题：编译错误 "LocalizationService 不存在"
**解决**：
1. 在 C# 文件顶部添加：`using SceneTodo.Services;`
2. 在 XAML 文件顶部添加：`xmlns:services="clr-namespace:SceneTodo.Services"`

### 问题：切换语言后没有变化
**解决**：
1. 确保点击了"保存"按钮
2. **重启应用**（语言切换需要重启）

### 问题：部分文本没有翻译
**解决**：
1. 检查是否遗漏了某些硬编码文本
2. 逐步迁移所有文本到资源文件
