### 主要变化说明

#### 1. 添加了命名空间（第5行）

```xaml
xmlns:services="clr-namespace:SceneTodo.Services"
```

#### 2. 标题国际化（第28行）

**之前**:
```xaml
<TextBlock Text="Tags"
```

**之后**:
```xaml
<TextBlock Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Tags_Title]}"
```

#### 3. 工具提示国际化

所有 `ToolTip` 属性都改为绑定：

```xaml
ToolTip="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Tags_NewTag]}"
```

#### 4. "使用次数"文本国际化（第139行）

**之前**:
```xaml
<TextBlock>
    <Run Text="Used: "/>
    <Run Text="{Binding UsageCount}" FontWeight="Bold"/>
</TextBlock>
```

**之后**:
```xaml
<TextBlock>
    <Run Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Tags_Used], Mode=OneWay}"/>
    <Run Text=" "/>
    <Run Text="{Binding UsageCount}" FontWeight="Bold"/>
</TextBlock>
```

## ?? 最佳实践

### 1. 命名规范

遵循前缀规则：
```
Common_     通用操作
Menu_       菜单项
Settings_   设置相关
Todo_       待办相关
Tags_       标签相关
Filter_     筛选相关
Message_    消息提示
Button_     按钮
Label_      标签
```

### 2. 绑定方式

**推荐**: 使用完整绑定
```xaml
Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[ResourceKey]}"
```

**可选**: 在 Run 中使用 Mode=OneWay
```xaml
<Run Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[ResourceKey], Mode=OneWay}"/>
```

### 3. 添加资源键的流程

1. 在 `Strings.resx` 中添加中文
2. 在 `Strings.en.resx` 中添加对应英文
3. 确保键名完全一致
4. 在 XAML 中使用绑定
5. 构建并测试

### 4. 测试步骤

1. 构建项目
2. 启动应用（中文）
3. 检查所有文本
4. 切换到英文
5. 重启应用
6. 再次检查所有文本
