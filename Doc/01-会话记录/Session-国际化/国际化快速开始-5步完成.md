## ?? 如果遇到问题

### 编译错误

**问题**: 提示重复定义  
**解决**: 
```
1. 停止调试 (Shift + F5)
2. 生成 → 清理解决方案
3. 删除 bin 和 obj 文件夹
4. 重新构建 (Ctrl + Shift + B)
```

### 文本没有翻译

**问题**: 切换语言后文本没有变化  
**解决**:
```
1. 确认资源文件已保存
2. 清理并重新构建
3. 完全重启应用（不是重新调试）
4. 检查资源键拼写
```

### 绑定错误

**问题**: 输出窗口显示绑定失败  
**解决**:
```
1. 检查是否添加了命名空间：
   xmlns:services="clr-namespace:SceneTodo.Services"
2. 检查绑定语法是否正确
3. 检查资源键是否存在于两个资源文件中
```
