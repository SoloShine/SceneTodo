## 技术细节

### 使用方式

#### 在 XAML 中
```xaml
<!-- 方法1: 通过 Binding -->
<Button Content="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_Save]}" />

<!-- 方法2: 通过 LocalizationExtension (推荐，但需要在每个文件中添加) -->
<Button Content="{services:Localization Common_Save}" />
```

#### 在 C# 中
```csharp
using SceneTodo.Services;

// 直接访问
string text = LocalizationService.Instance["Common_Save"];

// 格式化字符串
string msg = LocalizationService.Instance.GetString("Welcome_{0}", userName);
```

### 命名空间
需要在 XAML 文件中添加：
```xaml
xmlns:services="clr-namespace:SceneTodo.Services"
```

## 迁移指南

### 快速迁移步骤

1. **添加命名空间**（XAML 文件）
   ```xaml
   xmlns:services="clr-namespace:SceneTodo.Services"
   ```

2. **替换硬编码文本**
   - 找到硬编码的中文/英文文本
   - 在资源文件中添加对应的键值对
   - 使用绑定替换

3. **测试验证**
   - 构建项目
   - 运行应用
   - 切换语言测试

### 常见模式

#### 按钮文本
```xaml
<!-- Before -->
<Button Content="保存" />

<!-- After -->
<Button Content="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_Save]}" />
```

#### 工具提示
```xaml
<!-- Before -->
<Button ToolTip="帮助" />

<!-- After -->
<Button ToolTip="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Menu_Help]}" />
```

#### 占位符文本
```xaml
<!-- Before -->
<TextBox hc:InfoElement.Placeholder="请输入..." />

<!-- After -->
<TextBox hc:InfoElement.Placeholder="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Input_Placeholder]}" />
```

## 注意事项

### 开发注意
1. **保持键名一致**: 中英文资源文件的键必须完全一致
2. **命名规范**: 使用分类前缀（Common_, Menu_, Settings_ 等）
3. **测试完整性**: 每次迁移后都要测试中英文显示
4. **避免重复**: 先检查是否已有相同含义的键

### 用户注意
1. **重启生效**: 语言切换需要重启应用
2. **默认中文**: 第一次启动默认使用中文
3. **自动检测**: 可选择自动检测系统语言
