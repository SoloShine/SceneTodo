## ?? 常见问题和解决方案

### 问题 1: 编译错误 - 重复定义
**解决**: 停止调试 → 清理解决方案 → 删除 obj 和 bin 文件夹 → 重新构建

### 问题 2: 文本没有翻译
**解决**: 检查资源键名称 → 确认资源文件已保存 → 清理并重新构建

### 问题 3: 绑定错误
**解决**: 检查命名空间 → 检查绑定语法 → 检查资源键是否存在

### 问题 4: 文件被锁定
**解决**: 停止调试 → 关闭所有 Visual Studio 实例 → 重启 Visual Studio
