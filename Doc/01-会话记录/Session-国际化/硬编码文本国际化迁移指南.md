### 步骤 1: 停止调试



### 步骤 3: 替换 BackupManagementWindow.xaml

#### 3.1 添加命名空间

在文件顶部，找到：
```xaml
          xmlns:converters="clr-namespace:SceneTodo.Converters"
          Title="Backup Management"
```

替换为：
```xaml
          xmlns:converters="clr-namespace:SceneTodo.Converters"
          xmlns:services="clr-namespace:SceneTodo.Services"
          Title="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Backup_Title]}"
```

#### 3.2 替换标题文本（第37行）

找到：
```xaml
<TextBlock Text="Backup Management"
```

替换为：
```xaml
<TextBlock Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Backup_Title]}"
```

#### 3.3 替换按钮文本

**创建备份**（第49行）：
```xaml
<!-- 原文 -->

<!-- 替换为 -->
<TextBlock Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Backup_CreateBackup]}"/>
```

