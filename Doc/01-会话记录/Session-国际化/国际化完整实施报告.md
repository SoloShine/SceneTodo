## ?? 使用示例

### 如何使用国际化

#### 在 XAML 中
```xaml
<!-- 添加命名空间 -->
<Window xmlns:services="clr-namespace:SceneTodo.Services">

<!-- 使用绑定 -->
<Button Content="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_Save]}" />
<TextBox hc:InfoElement.Placeholder="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Search_Placeholder]}" />
</Window>
```

#### 在 C# 中
```csharp
using SceneTodo.Services;

// 获取文本
string text = LocalizationService.Instance["Common_Save"];

// 用于消息
HandyControl.Controls.Growl.Success(LocalizationService.Instance["Message_SaveSuccess"]);
```

### 访问语言设置
1. 点击主窗口左下角的"设置"按钮
2. 选择"语言设置"菜单项
3. 在弹出的窗口中选择语言
4. 点击"保存"
5. 重启应用生效

## ?? 迁移最佳实践

### 迁移流程

1. **准备阶段**
   - 查看文件，列出所有硬编码文本
   - 检查是否已有相应的资源键
   - 如没有，先添加到资源文件

2. **添加资源键**
   ```
   - 打开 Resources/Strings.resx（中文）
   - 添加键值对，如: Todo_Title = 标题
   - 打开 Resources/Strings.en.resx（英文）
   - 添加对应键: Todo_Title = Title
   ```

3. **替换文本**
   - 在 XAML 顶部添加命名空间（如未添加）
   - 用绑定替换硬编码文本
   - 在 C# 中用 LocalizationService 替换字符串

4. **测试验证**
   - 构建项目
   - 运行应用
   - 切换语言测试

### 命名规范

- **Common_**: 通用操作（保存、取消等）
- **Menu_**: 菜单项
- **Settings_**: 设置相关
- **Todo_**: 待办项相关
- **Filter_**: 筛选相关
- **Message_**: 消息提示
- **Error_**: 错误信息
- **Label_**: 标签文本

## ?? 技巧和提示

### XAML 绑定技巧

```xaml
<!-- 简单文本 -->
<Button Content="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_Save]}" />

<!-- 带参数的模板 -->
<Button>
    <Button.ContentTemplate>
        <DataTemplate>
            <StackPanel Orientation="Horizontal">
                <iconPacks:PackIconMaterialDesign Kind="Filter" Margin="0,0,4,0"/>
                <TextBlock Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Filter_Button]}"/>
            </StackPanel>
        </DataTemplate>
    </Button.ContentTemplate>
</Button>
```

### C# 使用技巧

```csharp
using SceneTodo.Services;

// 简单文本
var text = LocalizationService.Instance["Common_Save"];

// 格式化文本
var msg = LocalizationService.Instance.GetString("Welcome_{0}", userName);

// 在消息框中使用
HandyControl.Controls.MessageBox.Show(
    LocalizationService.Instance["Message_DeleteConfirm"],
    LocalizationService.Instance["Message_Confirm"],
    MessageBoxButton.YesNo);
```
