## [1.0.0] - 2024 - 国际化基础框架

### ? 新增功能
- **国际化服务**: LocalizationService 单例服务
- **语言枚举**: SupportedLanguage (中文简体/英文)
- **语言设置模型**: LanguageSettings 集成到 AppSettings
- **语言设置窗口**: LanguageSettingsWindow 独立窗口
- **资源文件**: Strings.resx (中文) 和 Strings.en.resx (英文)
- **自动检测**: 支持自动检测系统语言
- **持久化**: 语言设置自动保存

### ?? 核心组件
1. **Services/LocalizationService.cs**
   - 单例模式
   - 索引器访问资源
   - 动态切换语言
   - 属性变更通知

2. **Models/SupportedLanguage.cs**
   - ChineseSimplified (中文简体)
   - English (英文)

3. **Models/LanguageSettings.cs**
   - CurrentLanguage 属性
   - AutoDetect 属性

4. **Views/LanguageSettingsWindow.xaml**
   - 语言选择单选按钮
   - 自动检测复选框
   - 保存/取消按钮

5. **Resources/**
   - Strings.resx (中文，默认)
   - Strings.en.resx (英文)
   - Strings.Designer.cs (自动生成)

### ?? 使用方式
```csharp
// C# 中
string text = LocalizationService.Instance["Common_Save"];

// XAML 中
<Button Content="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_Save]}" />
```

## 贡献指南

### 添加新语言
1. 在 `SupportedLanguage.cs` 添加新枚举
2. 创建新资源文件（如 `Strings.ja.resx`）
3. 翻译所有资源键
4. 在 `LocalizationService.cs` 添加 CultureInfo 映射
5. 更新 `LanguageSettingsWindow.xaml`

### 添加新资源键
1. 在 `Strings.resx` 添加中文键值对
2. 在 `Strings.en.resx` 添加英文翻译
3. 遵循命名规范（Common_, Menu_, Settings_ 等）
4. 更新相关文档

### 提交规范
- feat: 新功能
- fix: Bug 修复
- docs: 文档更新
- refactor: 代码重构
- i18n: 国际化相关
