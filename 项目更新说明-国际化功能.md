## ?? 修改的文件 (2个)

### 1. **Models/AppSettings.cs**
```csharp
// 添加了语言设置属性
public LanguageSettings Language { get; set; } = new LanguageSettings();

// 在 Reset() 方法中添加了语言设置的重置
Language = new LanguageSettings();
```

### 2. **App.xaml.cs**
```csharp
// 添加了 using
using SceneTodo.Models;

// 在 OnStartup 方法中添加了语言初始化
protected override async void OnStartup(StartupEventArgs e)
{
    base.OnStartup(e);
    await InitializeDatabaseAsync();
    InitializeLanguage();  // <- 新增
    // ... 其他代码
}

// 新增方法：InitializeLanguage()
private static void InitializeLanguage()
{
    try
    {
        var settings = AppSettings.Load();
        
        if (settings.Language.AutoDetectLanguage)
        {
            LocalizationService.Instance.AutoDetectLanguage();
        }
        else
        {
            LocalizationService.Instance.ChangeLanguage(settings.Language.CurrentLanguage);
        }
        
        System.Diagnostics.Debug.WriteLine($"Language initialized: {LocalizationService.Instance.CurrentCulture.Name}");
    }
    catch (Exception ex)
    {
        System.Diagnostics.Debug.WriteLine($"Failed to initialize language: {ex.Message}");
        // 使用默认语言（中文）
        LocalizationService.Instance.ChangeLanguage(SupportedLanguage.ChineseSimplified);
    }
}
```

## ?? 使用方法

### 在 XAML 中使用

```xaml
<!-- 添加命名空间 -->
<Window xmlns:services="clr-namespace:SceneTodo.Services">
    
    <!-- 方法1: 使用 Localization 扩展（推荐） -->
    <Button Content="{services:Localization Common_Save}" />
    
    <!-- 方法2: 使用标准 Binding -->
    <TextBlock Text="{Binding Source={x:Static services:LocalizationService.Instance}, Path=[Common_OK]}" />
    
</Window>
```

### 在 C# 中使用

```csharp
using SceneTodo.Services;
using SceneTodo.Resources;

// 方法1: 使用强类型辅助类（推荐）
string text = Strings.Common_Save;

// 方法2: 使用服务
string text = LocalizationService.Instance["Common_Cancel"];

// 方法3: 格式化字符串
string msg = LocalizationService.Instance.GetString("Welcome_{0}", userName);
```

### 切换语言

```csharp
using SceneTodo.Models;
using SceneTodo.Services;

// 切换到英文
LocalizationService.Instance.ChangeLanguage(SupportedLanguage.English);

// 切换到中文
LocalizationService.Instance.ChangeLanguage(SupportedLanguage.ChineseSimplified);

// 自动检测
LocalizationService.Instance.AutoDetectLanguage();
```

### 打开语言设置窗口

```csharp
using SceneTodo.Models;
using SceneTodo.Views;

var settings = AppSettings.Load();
var window = new LanguageSettingsWindow(settings);
window.ShowDialog();
```

## ?? 配置说明

### 默认设置
- **默认语言**: 中文（简体）
- **自动检测**: 关闭（可在设置中启用）
- **设置位置**: `%LocalAppData%\SceneTodo\settings.json`

### 语言设置示例
```json
{
  "Language": {
    "CurrentLanguage": "ChineseSimplified",
    "AutoDetectLanguage": false
  }
}
```

## ? 常见问题

**Q: 如何添加新的资源字符串？**
A: 在 Visual Studio 中打开 `Resources/Strings.resx`，添加键值对，然后在 `Strings.en.resx` 中添加对应的英文翻译。

**Q: 语言切换后需要重启应用吗？**
A: 是的，目前需要重启才能完全生效。未来可以优化为动态切换。

**Q: 如何添加新语言？**
A: 1) 在 `SupportedLanguage` 枚举中添加；2) 创建对应的 resx 文件；3) 在 `LocalizationService` 中添加处理逻辑。

**Q: 资源文件在哪里修改？**
A: 在 Visual Studio 中打开 `.resx` 文件，会有可视化编辑器。
